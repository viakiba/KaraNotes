### 项目准备介绍

       其中：db.properties     数据库配置文件
         ehcache.xml       ehcache缓存配置

         其他文件介绍略

### 项目介绍
```html
java语言结合Spring以及SpringMvc,Mybatis框架实现的一个移动端博客项目。
并且使用DRUID实现数据库连接池设置，以及ehcahce实现缓存配置。（缓存还有很大的优化空间）
```
